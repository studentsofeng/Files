Exercise 1: Fibonacci Numbers

The sequence of Fibonacci numbers is 1, 1, 2, 3, 5, 8, 13, 21, 34, ... Fn where the sequence starts with two 1s and every subsequent Fibonacci number is the sum of the previous two (Fn+1 = Fn + Fn-1; for example, 21 = 8 + 13). Sometimes 0 is included as the initial element: F0 = 0.

Write a program which uses a while loop to calculate and print n and Fn, as in the following sample output. Print all Fibonacci numbers less than 1030.

n Fn 1 1 2 1 3 2 4 3 5 5 6 8 7 13 8 21 9 34 10 55 11 89 12 144 ...

When you first try this task, coding the loop can be surprisingly tricky. Your loop will have to keep track of the previous two Fibonacci numbers in order to add them together. For this aim you may use some temporary variables to store the previous Fibonacci numbers in them. (It greatly simplifies the code if you use tuple assignment, where two assignments are done simultaneously. (We will discuss it in the class later).

Note that the numbers you print become large enough to exceed the maximum possible integer value in other programming languages, and the maximum number of digits of precision possible with floats.

Exercise 2: Fibonacci Identities and Summing

Fibonacci numbers are fascinating to mathematicians and hobbyists because they have so many simple properties. For example, the sum of the first n Fibonacci numbers is itself one less than Fn+2. To check this property, modify the code from Exercise 1 to sum the Fibonacci numbers you calculate and print the total out too, like this (make sure to leave room to line up the columns):

n Fn Sum of Fn 1 1 1 2 1 2 3 2 4 4 3 7 5 5 12 6 8 20 7 13 33 8 21 54 [one less than F10] 9 34 88 10 55 143 ...

Exercise 3: Root Approximator

The Heron method for computing square roots starts with a guess and refines it. If x is a guess for the square root of a, then the average of x and a/x is a better guess. Write a loop that starts with a guess of x is 1, and then calculates guesses (averages) until the difference between successive guesses is within 10-6.

Once you are confident that your loop works, nest it within another loop that will print out the Heron-estimated square roots and the math.sqrt of the following values: 0.3, 0.5, 0.9, 1.7, 3.3, etc. up to 819.3 (start at 0.3, and add 0.2, then 0.4, then 0.8, etc.).

The structure of the code should look something like this:

aa, diff = ?, ? # number to find root of, and difference between numbers while aa < 820 : xx = ? while abs(...) > 1.e-6 : xx = ...
print ("square root of ..." diff = ? aa = ?
