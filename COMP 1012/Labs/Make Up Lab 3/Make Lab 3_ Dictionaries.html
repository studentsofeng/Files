Exercise 1: Dictionary Exercise

A dictionary is like an array where each stored key has a value associated with it. Integers and strings make good keys. Type import this into your Python shell to see the poem called "The Zen of Python" , by Tim Peters. Perform the following steps:

  • Copy and paste the text of this poem into a script file, and assign it to a variable, using triple quotes to enclose it.
  • Convert it to lower case (using .lower()), and split the text into a list of words. Print the list.
  • Create a new dictionary (e.g., dictWords = {}). Loop over the words in the list, and count the number of times each one appears. That is, for each word, consult the entry in dict with that key (dictWords[word]). If it doesn't exist (word not in dictWords), create it and give in an initial value of 1 (e.g., dictWords[word] = 1). If it does already exist, increase its value by 1.
  • Print the unique words of the poem (dictWords.keys()).

Exercise 2: Dictionary manipulation

Using the same poem as before, create a dictionary that counts how many words start with each letter.

  • Create another new dictionary (e.g., letterCounts= {}).
  • Loop over the words in the list, and count the number of words that start with each letter.
  • Find the most frequent starting letter for words in the poem, by using a loop to find the maximum value in the dictionary.

Exercise 3: Create a bar chart

This material is not testable, and is enrichment only. Using the data from the lettersCounts dictionary, create a bar chart using matplot lib. An example is shown on the right.

  • Access an subplot using
    fig, ax = mpl.subplots()
  • Set a bar chart using the ax.bar function. It takes three parameters:
    1. A list or numpy array of the x-coordinates of the left ends of the bars
    2. A list or numpy array of the heights of each of the bars.
    3. A floating point number giving the width of each bar.
  • Create x axis labels (ticks) with the following two commands:
    1. ax.set_xticks that takes a single parameter, which is a list or numpy aray of the x-coordinates of the centres of the bars, where the labels will be placed.
    2. ax.set_xticklabels that takes a single parameter, which is a list or numpy array of the labels for the bottoms of the bars.
