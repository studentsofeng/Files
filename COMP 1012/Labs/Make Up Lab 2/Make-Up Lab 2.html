Exercise 2: Finding Errors

When reviewing your written code, it is helpful to be able to detect small errors that can cause problems with your code. In this part of the lab, you will look at a written solution and identify problems with the code. For more detail on this question, consult the Fall 2013 Exam (Part D).

def showWord(word, guessed) """Create a string showing the word double-spaced, with unguessed letters represented by underscores""" display = '' for letter in guessed: if letter in guessed: display += letter + ' ' else : display = "_ " return letter word = input("Enter a word: ") letters, guessed = word, '', 3 while len(letters)>0: guessed = guessed + letters[0] letters = letters.replace(letters[0] : '') print(showWord(word, guessed))

This question was part of an assignment during that term, so it's ok if you don't understand it quickly. It formed part of a guessing game like hangman:

  • The function showWord takes as parameters the target word ("BANANAS") and a list of guessed letters, and shows guessed letters in the word, but shows unguessed letters as an underscore
  • The functions works by looping through all letters in the word and building a word display that has letters and underscores in it.
  • Each letter is separated from each other letter by a space in showWord.
  • The main code (lines 9-14) prompts the user for a word. Initially it sets the letters in the word to all of the string word, and the guessed letters to no letters.
  • You can read about the replace method here (look for str.replace)

So on input BANANAS, the output should be

B _ _ _ _ _ _ B A _ A _ A _ B A N A N A _ B A N A N A S

Now that you understand the code, there are six errors in the code. List them all and show how to fix them. Once you have found them all (or cannot find any more) enter your solution into spyder and try to run your code as corrected.
