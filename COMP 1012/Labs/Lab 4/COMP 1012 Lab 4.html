Exercise 1: Tuple warm-up

  2. Create another tuple representing a vector from A to B (that is, B-A). Notice you can't just subtract one from the other; tuples don't support subtraction. Perform the subtraction one coordinate at a time by subscripting in the original tuples. Print the vector. What is wrong with the following code, if ptBminusA is the difference vector tuple? Exercise 3: Creating lists with a while loop

Use either one or two while loops to generate the same list that you produced in step 5 of the previous exercise (odd numbers counting up and down) by counting through the values; use listY.append to add each value to the end.

Try to do it with a single loop. To make a single loop count up and then down, use a boolean variable called increasing. Each time through the loop, set increasing to True if you have fewer than 10 (or HALFCOUNT) entries, and False thereafter. Rely on the knowledge that True is an alias for 1 and False is equivalent to 0 to come up with expressions that will either increase or decrease the loop counter. If you know what an "if statement" is, do not use it! Hint: you could use a short list of possible increments, or you could use a power (and many other expressions).
