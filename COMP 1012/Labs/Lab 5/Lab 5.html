Exercise 1: Printing shapes.

In this exercise you will write a Python script that will print out several different shapes. In each case, create the shape row by row, using one for loop to advance the rows, and one or more for loops to add the characters in a row to the end of the current string. Make your code work for several different sizes, including the one specified in each case. Since this is an exercise in using nested for loops, you are not allowed to use 5*'X' nor any if statements.

Here are the figures to print:

  1. An empty square of size 8 using "*" :
    A HOLLOW SQUARE OF SIZE 8 ******** * * * * * * * * * * * * ********
    Hmmm... It doesn't look much like a square, does it? The space allocated to each character in this font is actually rectangular, so an nÃ—n array of characters is rectangular rather than square. To make the exercise more challenging, you could figure out a correction factor by which you multiply the width to make your designs look more like squares. But first try these exercises without this complication.

    Here's an example of the code to produce this square (without any corrections to make it look more square). The variable row is assigned the number of each row, but it is not actually used for anything. It is basically a counter. Note how all the characters to print a row of the the hollow square are held by a string variable text.

    You'll have to figure out the rest of the examples for yourself.

    SIZE = 8 ; CHAR = '*' # This code will work for a wide range of SIZE values # top row is solid for row in [ 0 ] : text = "" for col in range(SIZE) : text += CHAR print (text) # middle rows have a left character, blanks, and right character for row in range(1, SIZE - 1) : text = CHAR for col in range(SIZE - 2) : text += ' ' text += CHAR print (text) # bottom row is solid for row in [ SIZE - 1 ] : text = "" for col in range(SIZE) : text += CHAR print (text)
  2. A solid right triangle of size 12 with 'T':

    Output a right triangle where the hypotenuse goes from the top right of the triangle to the bottom left using the dimension 12 and character 'T'. Remember: your code should work for other sizes too if you change the value assigned to SIZE, and for other characters if you change what you assign to CHAR

    A SOLID RIGHT TRIANGLE OF SIZE 12 TTTTTTTTTTTT TTTTTTTTTTT TTTTTTTTTT TTTTTTTTT TTTTTTTT TTTTTTT TTTTTT TTTTT TTTT TTT TT T
  3. A solid mirrored right triangle of size 9 with "$":

    Output a right triangle where the hypotenuse goes from the top left of the triangle to the bottom right using the dimension 9 and character '$':

    A SOLID MIRRORED RIGHT TRIANGLE OF SIZE 9 $$$$$$$$$ $$$$$$$$ $$$$$$$ $$$$$$ $$$$$ $$$$ $$$ $$ $

Exercise 2: Defining a function to print a shape.

The essence of programming is to write code that can solve problems where you don't know all the details in advance. Once your code gets the details, it should be able to come up with an answer by adjusting automatically to those details. We often code this self-adjusting code in a function with parameters. When other code calls the function, that code must provide arguments, which are values that are matched one-by-one to the corresponding parameters. The arguments provide the missing details, and the function solves the problem that is now fully specified.

For example, the following function will draw a hollow square of a specified size, using whatever character you want. Copy the code from this webpage and paste it into Anaconda to create the function. A call to drawSquare(8,'*') should produce the same results as the example above. Try it for several sizes, including 0, 1, 2, and 3. Can you figure out how it handles these small cases, when the code has three for loops, and looks like it should always print at least three rows? What happens when you pass an argument with several characters as the second argument? How would you fix it?

def drawSquare(SIZE, CHAR) : # top row is solid for row in range(1) : text = "" for col in range(SIZE) : text += CHAR print (text) # middle rows have a left character, blanks, and right character for row in range(1, SIZE - 1) : text = CHAR for col in range(SIZE - 2) : text += ' ' text += CHAR print (text) # bottom row is solid for row in range(max(SIZE - 1, 1), SIZE) : text = "" for col in range(SIZE) : text += CHAR print (text)
