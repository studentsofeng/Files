Exercise 4: Running programs from class

The Week 1 class notes are available as a pdf file on the course website. Try one of the computation exercises described in those notes:

  • estimate the volume of the Great Pyramid of Ghiza;
  • estimate the force of a jet of water on a plate;

With these problems, try the following:

  • evaluate some of the formulas shown in interactive mode (in the IPython console window), line by line, to see what the interface does;
  • write instructions for each problem in a separate script file (edit it in the left window and save it with a file name that ends with ".py"), and run the file (green triangle icon), fixing any errors that are detected so that it produces the right results;
  • write the code on your own without referring to the solutions in the notes, and check those solutions to compare with your results;
  • modify the data to solve similar but different problems, and modify the output statements to make the output you would like to see.

Add comment lines similar to the following to the beginning of your script files. We will require documentation of this type in all your assignments.

""" PyramidVolume COMP 1012 SECTION A01 or A02 INSTRUCTOR: Domaratzki or Boyer ASSIGNMENT: Lab 1, Exercise 4 AUTHOR: [your name] VERSION: 2016-Sep-14 PURPOSE: Calculate the volume of the Great Pyramid of Ghiza and display the volume in the shell window. """

Save the file to the H: (Home) drive on your lab computer if you can; otherwise use the desktop or a USB drive. You may have material stored on the "old" H: drive if this isn't your first year as a student. You will see instructions on the computer Desktop about retrieving that material and moving it to the "new" H: drive.

The desktop on a lab computer is stored on the computer itself, not on your network drive, so you should copy your files to a more permanent location before you leave. A good approach is to carry a thumb drive and plug it into the lab computer, using it to store your programs. Remember to take it with you!
